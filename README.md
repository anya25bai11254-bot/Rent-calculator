# Rent-calculator

Overview:

The Rent Calculator is a simple and useful Python project designed to help students, roommates, and flatmates calculate their monthly shared expenses easily.
Many people living in hostels or shared apartments struggle every month to divide rent, food cost, and electricity bills fairly. This project provides a fast, accurate, and convenient solution.

Purpose of the Project:

The main purpose of this project is to automate the monthly calculation of shared expenses.
Instead of manually calculating the rent split using a calculator or mobile app, this program computes:

Total electricity bill
Combined monthly expenses
Exact amount each person has to pay
This reduces errors and saves time.

Why This Project is Needed?

Students in hostels share rent, food, and electricity costs
Roommates usually calculate contributions manually
Manual calculations often lead to:

Mistakes
Confusion
Mismanagement

This project solves these problems by providing a fair, fast, and accurate calculation system.
 
Features:

Takes user input for all expenses:

Hostel/flat rent
Food expenses
Electricity units consumed
Charge per unit
Number of persons

Automatically calculates:

Electricity bill
Total monthly cost
Per person contribution
 
How It Works:

The user enters all the required cost details.

The program calculates the electricity bill using:
electricity_bill = units * charge_per_unit

Total expenses are added:
total_cost = rent + food + electricity_bill

The total is divided equally among all persons.

The final per-person amount is displayed on the screen.

Use Cases:

Hostel roommates splitting monthly rent
PG students sharing their electricity and food expenses
Flatmates dividing bills fairly
Anyone who wants simple cost-sharing without confusion

Benefits:

Saves time
Completely accurate
Eliminates manual calculation errors
Helps maintain transparency among roommates
 
Conclusion:

The Rent Calculator project is a practical and beginner-friendly Python application that solves a very real problem faced by students and shared-living groups.
It improves financial clarity, avoids confusion, and makes monthly expense sharing smooth and fair.
